# Agent Best Practices

This document outlines best practices for agents working on this project.

## Project Overview

This project is a viewer for CMake Presets files. The goal is to create a tool that can parse and display the contents of `CMakePresets.json` and `CMakeUserPresets.json` files in a user-friendly way.
